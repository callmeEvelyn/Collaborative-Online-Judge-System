# Collaborative-Online-Judge-System

### Steps to run the application

* You can simply run:

````

sh runner.sh

````

* Or

1. under your dir :

````
npm install
````

2. under dir/oj-client:

````
ng build --watch
````
3. under dir/oj-server:

```
node server.js
```
or
```
nodemon server.js
```

Then you can check http://localhost:3000 to see the collaborative online judge website.
